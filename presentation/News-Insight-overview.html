Every article is read by AI and turned into data.

Meaning

The text becomes a numeric fingerprint (an embedding), so similar articles sit close together in vector space.

multilingual-e5 · vector(384)

Mood

A language model labels the tone of each article.

positive neutral negative

Topic

Articles are grouped into recurring topics automatically, with no preset list, then each cluster is named.

BERTopic clustering

Region

Place names in the text are matched to one of Tunisia's 24 governorates.

gazetteer · Sfax · Tunis…
Why it matters: free-form text becomes structured signals you can search, chart and map.
One database holds the text and its meaning together.
Each article and its 384-dim vector live in the same PostgreSQL row, so "find similar articles" is a single SQL query with pgvector, not a second system to run.
pgvector (our choice)A dedicated vector database
InfrastructureNone extra: it is just PostgresA second system to run and keep in sync
ConsistencyOne transaction for data and vectorsTwo stores that can drift apart
Scale ceilingMillions of rows (with an index)Billions, very high query rates
Best forThis project's sizeMassive, high-traffic retrieval
The trade-off: at this corpus size, one database beats running two and syncing them. We would switch only if the corpus grew by orders of magnitude.

Why RAG (our choice)

Answers about today's articles with no retraining, and every claim links back to a source. Cheap: one embedding plus one prompt.

Why not fine-tuning

It would need a costly retrain for every new article, can still invent facts, and cannot cite where an answer came from.

It runs itself, records what it does, and rebuilds with one command.

Orchestration

A Prefect flow runs collect → transform → load on a daily schedule or on demand, with automatic retries.

Prefect · idempotent (upsert)

Tracking

Every clustering run's settings, metrics and fitted model are recorded, so any run can be reproduced.

MLflow · params · metrics · model

Monitoring

Each run appends one row: articles ingested, topics found, outlier rate, sentiment split.

pipeline_metrics

Reproducibility

The whole stack (database, API, app) starts together on any machine with Docker.

docker-compose up
Why it matters: orchestration, tracking, monitoring and reproducibility are what make this an operated platform, not a one-off notebook.
