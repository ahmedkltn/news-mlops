Retrieval-augmented generation (RAG): the model only sees the retrieved articles, so every answer is grounded in the corpus and can cite exactly where it came from.
Simple to run: one table, one command, all open-source.

What this project demonstrates.
1
A complete end-to-end platform
From live news sites to a searchable, answerable web app, running today.
2
Search by meaning
Semantic retrieval over French and Arabic articles, using pgvector and multilingual embeddings.
3
An assistant that cites its sources
Retrieval-augmented answers grounded in the corpus, so nothing is invented.
4
Reproducible MLOps
A scheduled, tracked and monitored pipeline; the whole stack starts with one command.
Current limits: four sources and French-first; Arabic dialect and more sources are the next step.
